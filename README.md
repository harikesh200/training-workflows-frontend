# Maintenance Workflow Console Frontend

Vite, React, and TypeScript frontend for the machine maintenance workflow API.
This project lives separately from the backend at
`D:\Training Workflows Frontend`.

```bash
pnpm install
pnpm dev
```

The frontend runs at `http://localhost:5173` and uses
`VITE_API_BASE_URL=http://localhost:3000` by default. See `.env.example`.
Run the backend from `D:\Training Workflows` with `bun run dev`.
